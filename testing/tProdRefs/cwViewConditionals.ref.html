If Then a boolean expression turns a block of code on or off
if (5 > 3)
	console.log("Yep, it is");

if (5 > 3) {
	console.log("Yep, it is");
	console.log("Ouais, il est")
}

just like in C 

if 5 > 3:
    print "Yep, it is"
    print "Ouais, il est"

Return your level of indent to close the block.

If Then Else a boolean expression chooses between two blocks of code
if (5 > 3)
	console.log("Yep, it is");
else
	console.log("no, it isn't");

if (5 > 3) {
	console.log("Yep, it is");
	console.log("Ouais, il est")
}
else {
	console.log("no, it isn't");
}

just like in C 

if 5 > 3:
    print "Yep, it is"
else:
    print "no, it isn't"
If Else If Else If a sequence of boolean expressions choose among several blocks of code
if (5 > 3)
	console.log("Yep, it is");
else if (5 > 4)
	console.log("Ouais, il est")
else
	console.log("no, neither are");

Note there is no explicit 'elseif' construct; it is just if statements as else conditions. 

if 5 > 3:
    print "Yep, it is"
elif 5 > 4:
    print "Yep, it is"
else:
    print "no, it isn't"


If Else Expressions a boolean expression chooses between two expressions
(5 > 3) ? "Yep, it is" : "no, it isn't"

"Yep, it is" if 5 > 3 else "no, it isn't"

Note the syntax follows english grammar, not the C style ? : operator.

Case Statements alternate values of an expression choose among several blocks of code
var res, x = 1.2, func = 'sin';
switch (func) {
case 'sin':
	x = Math.PI/2 - x;
case 'cos':
	res = Math.cos(x);
	break;

case 'tan':
	res = Math.tan(x);
	break;

default:
	 throw "Error: function "+ func +" not found";
}

Case Calculation what kind of variables and values are allowed to switch based upon?

Case variables can be primitives: booleans, numbers, and strings, and the undefined value.

Equality is determined with the === strict equality operator. For instance, '5' !== 5 , although 5.0 == 5 .
