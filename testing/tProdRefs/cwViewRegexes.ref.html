You need to import the re package to use regexes. 

import re
Regex is an Object? is a regex a distinct value and data type in this language, or is it just a character string with special syntax?

A regular expression as a string, preferably a raw string, without delimiters inside the string: 

re.search(r'\s*<br\s*/?>', inbuf)

As a raw string, you'll do less messing around because usually you can just enter one backslash instead of two: 

re.search('\\s*<br\\s*/?>', inbuf)

If you use it more than a few times, you should compile it into a re.RegularExpression-class object: 

regX = re.compile(r'\s*<br\s*/?>')
regX.search(inbuf)
Regex Detect decide if a match exists at start of target string

use brackets to detect if a substring is present: 

'panama'[/a.a/]
= 'a.a'   # true
'panama'[/pie/]
= nil



re.match('p.n', 'panama')
= (Match Object)

Returns true if there was a match at the start of the string, None otherwise.

Regex Search scan target string for matches

returns the actual (first) matching substring: 

'panama'[/a.a/]
= 'ana'

'panama'.index(/a.a/) 

= 1

mo = re.search('a.a', 'panama')
mo.start
= 1

Returns None if there was no match.

Regex Replace search for matches and replace with strings or templates

brackets will only replace the first occurrence 

s = 'help resolve this solve problem'
s[/r?e?solve/] = 'rectify'
s= 'help rectify this solve problem'

no match, no replacement 

s = 'help resolve this problem'
s[/avoid/] = 'sidestep'
s= 'help resolve this problem'

sub() and gsub() replace one or all matches 

"ethane\n".sub(/(\w+)ane/, '\1ane \1ylene \1anol')
= 'ethane ethylene ethanol'
"methane ethane propane".gsub(/(\w+)ane/, '\1ane \1ylene \1anol')
= 'methane methylene methanol ethane ethylene ethanol propane propylene propanol'

sub!() and gsub!() modify a string in-place 

s = "ethane\n"
s.gsub!(/(\w+)ane/, '\1ane \1ylene \1anol')
s= 'ethane ethylene ethanol'


re.sub() replaces one or more match, returning the result 

re.sub(r'(\w+)ane', r'\1ane \1ylene \1anol', "ethane")
= 'ethane ethylene ethanol'

The count allows multiple replacements. The default, 0, is to do all of them. 

re.sub(r'(\w+)ane', '\1anol', "methane&ethane&propane", 2)
= 'methanol&ethanol&propane'

The flags go last. 

re.sub(r'(\w+)ane', '\1anol', "METHANE&ETHANE&PROPANE", 0, re.I)
= "METHanol&ETHanol&PROPanol"


Loop by Regex execute a code block once per match

sub() and gsub() will replace matches with block returns; the result is the newly reconstructed string. sub() will do the first substitution only, gsub() will do all. They both have non-block forms that simply substitute a replacement string. 

'The rain plainly falls mainly in Spain'.gsub(/(\w)(\w)(\w*)/) {|match| $2 + $1 + $3 }
= 'hTe arin lpainly aflls aminly ni pSain'
'The rain plainly falls mainly in Spain'.sub(/(\w)(\w)(\w*)/) {|match| $2 + $1 + $3 }
= 'hTe rain plainly falls mainly in Spain'

scan() simply executes the block once for each match. scan() also has a non-block form that returns an array of matches. The original string and its unmatched parts are left behind. 

'The rain plainly falls mainly in Spain'.scan(/(\w)(\w)(\w*)/) {|c1, c2, rest| print c2, c1, rest, ' ' }
=> 'hTe arin lpainly aflls aminly ni pSain'

The code block allows you to do things that would be difficult or impossible with a simple regex replace. 

"ethane propane butane".gsub(/(\w+)ane/) {|match|
    if match == 'ethane' then 'acetylene' else $1 + 'yne' end
}
= 'acetylene propyne butyne'

You can use the sub() method - just pass a function instead of a replacement string. 

def pickupImage(mo):
    ft = FaceThumbnail(mo.group(1))

re.sub(r'<img .*href="?[^ "]+"?.*>', html, pickupImage, 0, re.I)


Split generate an array of strings by splitting the target at matches

note trailing empties gone 

'http://www.nps.gov/manz/'.split(/[:/.]/)
= [ 'http', , , 'www', 'nps', 'gov', 'manz']

second argument limits breaking 

'http://www.nps.gov/manz/'.split(/[:/.]/, 2)
=  'http', '//www.nps.gov/manz/' ]

note trailing empties included 

'http://www.nps.gov/manz/'.split(/[:/.]/, -1)
= [ 'http', , , 'www', 'nps', 'gov', 'manz', ]

re.split(r'[:/.]', 'http://www.nps.gov/manz/')
= [ 'http', , , 'www', 'nps', 'gov', 'manz']

third argument limits breaking 

re.split(r'[:/.]', 'http://www.nps.gov/manz/', 2)
= [ 'http', , , 'www', 'nps', 'gov', 'manz']

mo = re.search(r'<INPUT NAME="(\w+)" ID="(\w+)" />', '<FORM><INPUT NAME="CITY" ID="SIGNUP_CITY" />')
mo.expand("document.getElementById(\2).style.display = 'none';")
= "document.getElementById(SIGNUP_CITY).style.display = 'none';"
mo.group()
= '<INPUT NAME="CITY" ID="SIGNUP_CITY" />'
mo.start()
= 6
mo.end()
= 44
mo.group(1)
= 'CITY'
mo.group(2)
= 'SIGNUP_CITY'
mo.end(2)
= 40

Additional Constructs beyond the Perl regex syntax, what other constructs can you use in this language's regexes?
  • (?:...) non-captured parentheses 
/(?:yahoo|google) (Sponsored|AdWords) \1/.test('yahoo AdWords AdWords')
= true

note \1 refers to (Sponsored|AdWords) as the (?:yahoo|google) is not counted

  • (?=...) qualifying suffix 
s = "Alvin, the kid with the teeshirt that says Alvin Ross and there's a picture.";
s.replace(/Alvin(?= Ross)/g, 'Albert')
= "Alvin, the kid with the teeshirt that says Albert Ross and there's a picture."

The construct (?= Ross) matches the expression ' Ross' after the end of the pattern, but is not included in the match string. Must appear at the end of the pattern.

  • (?!...) disqualifying suffix 
s = "Alvin, the kid with the teeshirt that says Alvin Ross and there's a picture.";
s.replace(/Alvin(?! Ross)/g, 'Albert')
= "Albert, the kid with the teeshirt that says Alvin Ross and there's a picture."

The construct (?! Ross) must NOT match after the end of the pattern. Must appear at the end of the pattern.

  • \0 matches a null character (as long as it isn't followed by more digits)
